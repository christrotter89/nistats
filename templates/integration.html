Using our API
How to connect your application to our data

NI Stat's uses Firebase to create a REST API that anyone can connect to. This allows you to use our data in your own apps and projects. As a result, your service will benefit from updates to our data.

Get started

Our REST API is exposed at the following URL. Response data will be in JSON.

https://nistats-63495.firebaseio.com/.json

You may enter the above URL in to your browser to obtain a JSON object of all chart data. However, you can also reference specific charts by referencing their ID such as /0, /1 and /2.

Most programming languages allow you to connect a REST APIs, make requests and manipulate the data from responses. To learn more about REST API's, visit Code Academy's Javascript course and their API courses.

Individual datasets

Each dataset has three top-tier objects.
Meta data
Contains information relating to the context of the dataset such as name, publisher and source.
Data
Actual data for each series. On our website these are plotted on charts and rendered to data tables.
Options
This information is only relevant our website, and relates mostly to chart configurations such as scales and tooltip filters.

Simply looking for an Excel download?

Each dataset has a data tab that allows for downloading in Excel, CSV and JSON formats.
